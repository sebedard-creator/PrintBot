Configuration Imprimante Niimbot

Guide de Connexion Windows : Port COM Virtuel

Pour que Python puisse communiquer avec l'imprimante sans instabilité, vous devez router le Bluetooth vers un port COM Windows.

1. Allumez votre imprimante Niimbot B1.
2. Ouvrez les Paramètres Windows > Bluetooth et appareils. Ajoutez l'imprimante (B1-XXXX). Si code PIN demandé : 0000 ou 1234.
3. Au bas de la page Bluetooth, cliquez sur Paramètres des appareils supplémentaires (ou Plus de paramètres Bluetooth).
4. Dans la nouvelle fenêtre, allez dans l'onglet Ports COM et cliquez sur Ajouter....
5. Cochez Sortant (l'ordinateur lance la connexion), sélectionnez votre imprimante B1-XXXX, puis cliquez sur OK.
6. Windows va créer une ligne du genre : Port : COM8 | Direction : Sortant. Notez ce numéro de port !
7. Sélectionnez ce numéro dans la liste ci-dessous et cliquez sur Tester.

Assurez-vous que l'imprimante est allumée et jumelée à Windows via les paramètres Bluetooth.
